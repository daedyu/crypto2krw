# Product Requirements Document (PRD): Crypto2KRW MVP

## 1. Project Overview
Crypto2KRW는 사용자가 보유한 암호화폐(SOL, USDT, ETH)를 원화(KRW) 가치로 실시간 환산하여, 오프라인 가맹점에서 QR 코드로 즉시 결제할 수 있게 해주는 모바일 결제 플랫폼이다.

## 2. Target Audience
- 암호화폐를 실생활에서 사용하고 싶은 크립토 홀더
- 초기 제휴를 맺은 자체 QR 가맹점 (팝업스토어, 특정 프랜차이즈 등)

## 3. Core Features (MVP)
### 3.1 User (Customer) App
- **회원가입 및 지갑 생성:** 이메일 가입 및 유저별 고유 입금 지갑(Deposit Address) 발급.
- **자산 조회:** 입금된 암호화폐 잔액 및 실시간 KRW 환산 가치 조회.
- **결제 우선순위 설정:** 결제 시 차감될 암호화폐 순위 설정 (예: 1. USDT, 2. SOL).
- **QR 결제 (User Scan):** 가맹점의 QR 코드를 스캔하여 결제 진행 (실시간 환율 적용 및 슬리피지 동의).
- **결제 내역:** 결제 이력 및 암호화폐 차감 내역 제공.

### 3.2 Merchant (가맹점) Web/App
- **결제 QR 생성:** 결제 금액(KRW)을 입력하여 1회용 결제 QR 코드 생성.
- **결제 완료 알림:** 유저가 결제 완료 시 실시간 승인 알림(WebSocket/Push).
- **정산 내역 조회:** 매출 내역 및 KRW 정산 예정 금액 조회.

## 4. Key Policies
- **결제 환율 (Oracle):** 결제 시점의 바이낸스/업비트 기준 환율에 수수료/슬리피지 마진(예: 1~2%)을 더한 내부 고시 환율 적용.
- **트랜잭션 속도:** 암호화폐 온체인 트랜잭션을 기다리지 않고, 서버 내부의 오프체인 장부(Off-chain Ledger)를 차감하여 1초 이내에 결제 승인.
